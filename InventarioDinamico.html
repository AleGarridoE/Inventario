Introduccin

Una matriz es un arreglo de nmeros en filas y columnas. Cada nmero dentro de una matriz corresponde a un elemento o entrada. Las dimensiones de una matriz es el nmero dado de filas y columnas en la matriz en ese orden.

Siempre y cuando las dimensiones de las matrices sean las mismas, es posible realizar las operaciones de suma y resta, que se llevan a cabo entrada por entrada.

La aplicacin de las matrices tiene un rol crucial en casi todos los campos de las ciencias exactas, ya que permite el planteamiento y solucin de sistemas de ecuaciones lineales.

El inventario de los productos de una compaa puede ser planteado como un sistema de matrices, ya que cada producto tiene multiples atributos, como el tamao, color, peso, talla, costo, material, procesos de acabado y empaquetamiento, por ejemplo. Un sistema bidimensional o tridimiencional podra representar todas las configuraciones posibles para el inventario de productos de una compaia.

Un sistema de inventario matricial simplifica el manejo de entradas y salidas de los productos en tiempo real, al permitir que cada atribo del producto sea asignado a las filas y columnas de una matriz.

El empleo de invendatios dinmicos requiere de el uso de algorimos computacionales que permita modificar en tiempo real las entradas de las matrices. Mediante el uso de recursos computacionales se optimiza la gestion de los productos de una compaa.

Justificacin

Un inventario dinmico permite introducir los productos en sistemas matriciales. Mediante la compra y venta de productos se lleva a cabo la suma y resta de matrices. Al asignar precios a lo productos y por medio la compra de productos se puede llevar a cabo la multiplicacion de escales.

El inventario de una tienda de zapatos permite la asignacion de variables a los renglones y columnas de cada matriz. Siendo as cada entrada de cada matriz un producto diferente.

El resumen del inventario permite ver el estado de las matrices antes y despues de la compra de productos y la adquicicion de zapatos por parte de los proveedores. El inventario de los productos permite ver si efectivamente se realizaron las operaciones seleccionadas.
